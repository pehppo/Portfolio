Passaporte Literário

Sobre o Projeto

O Passaporte Literário é um aplicativo desenvolvido em Flutter que permite aos usuários registrar e acompanhar suas leituras de forma organizada e visual.

Funcionalidades

  • Adicionar e editar livros lidos
  • Diário de leitura
  • Metas e estatísticas
  • Integração com Google Books API

Tecnologias Utilizadas

Flutter Dart Google Fonts Shared Preferences Image Picker HTTP
